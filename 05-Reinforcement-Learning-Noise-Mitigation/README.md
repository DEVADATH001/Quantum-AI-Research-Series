# Project 05: Quantum Policy Gradient with Noise Simulation

## Objective

Train a simple quantum policy model in a toy environment, then evaluate performance under noise and basic mitigation.

## Stack

- Python
- NumPy
- Matplotlib
- Qiskit (+ Aer noise simulation)
- Gymnasium (optional simplified environment)
- Jupyter Notebook

## Notebook

- `project_05_quantum_rl_noise.ipynb`

## Tasks

1. Define a simple environment (2-state or custom toy control).
2. Build parameterized quantum policy circuit.
3. Implement policy gradient update loop.
4. Train with noiseless and noisy simulators.
5. Apply simple mitigation and compare reward curves.

## Expected Outputs

- Reward vs episode curves (clean/noisy/mitigated).
- Discussion of noise impact and mitigation limits.
