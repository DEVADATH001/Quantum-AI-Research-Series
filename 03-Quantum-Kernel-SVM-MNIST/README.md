# Project 03: Quantum Kernel SVM for Image Classification

## Objective

Compare a classical SVM and a QSVM-style quantum-kernel approach on a small, reduced-dimensional image dataset.

## Stack

- Python
- NumPy
- Matplotlib
- scikit-learn
- Qiskit
- Qiskit Machine Learning
- Jupyter Notebook

## Notebook

- `project_03_qsvm_vs_classical.ipynb`

## Tasks

1. Load a small image dataset subset.
2. Reduce dimensions aggressively (PCA).
3. Train classical SVM baseline.
4. Build quantum feature map + kernel classifier.
5. Compare accuracy and runtime.

## Expected Outputs

- Baseline classical SVM metrics.
- Quantum kernel classifier metrics.
- Side-by-side comparison table/plot.
